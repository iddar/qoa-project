# 01-overview

Esta carpeta contiene la documentación de visión, alcance y glosario del proyecto.

## Cómo rellenar

### Visión
- Crea un archivo `vision.md` con:
  - Objetivo del producto (1 página resumida).
  - Qué sí hace y qué NO hace el sistema.
  - Stakeholders principales y casos de uso clave.

### Alcance
- Crea un archivo `alcance.md` con:
  - Límites del proyecto.
  - Funcionalidades incluidas y excluidas.
  - Criterios de aceptación.

### Glosario
- Crea un archivo `glosario.md` con:
  - Términos técnicos y definiciones.
  - Abreviaturas y siglas utilizadas en el proyecto.
