# 05-security

Esta carpeta contiene la documentación de seguridad, incluyendo autenticación, permisos y modelo de amenazas.

## Cómo rellenar

### Autenticación y Permisos
- Crea `auth-permisos.md` con:
  - Mecanismos de autenticación (OAuth2, JWT, API keys).
  - Sistema de permisos y roles.
  - Gestión de accesos.

### Modelo de Amenazas
- Crea `threat-model.md` con:
  - Análisis de riesgos (auth, permisos, fuga de datos, abuso de APIs).
  - Mitigaciones.

### Estándares y Cumplimiento
- Crea `estandares.md` con:
  - OWASP ASVS y Top 10.
  - Logging y auditoría.
  - Gestión de secretos (Vault/KMS).
  - Cifrado en tránsito y reposo.

### Privacy
- Crea `privacy.md` con:
  - Clasificación de datos (PII, sensibles).
  - Retención y borrado.
